# Image Setup Instructions

To display your photos on the website, please copy your image files into this folder:
`src/assets/images/`

Rename your photos to match the following filenames:

### Main Images
- `hero.jpg` - Main banner image (Home page)
- `about-bg.jpg` - Background for About page header
- `cta-bg.jpg` - Background for the "Start Journey" banner

### Programs (Service Cards)
- `program-1.jpg` - Strength Training
- `program-2.jpg` - CrossFit / HIIT
- `program-3.jpg` - Cardio
- `program-4.jpg` - Weight Loss
- `program-5.jpg` - Group Classes
- `program-6.jpg` - Personal Training

### Trainers
- `trainer-1.jpg` - Trainer 1
- `trainer-2.jpg` - Trainer 2
- `trainer-3.jpg` - Trainer 3
- `trainer-4.jpg` - Trainer 4

*Note: If you don't have enough photos, you can reuse the same photo by copying it and giving it multiple names.*
