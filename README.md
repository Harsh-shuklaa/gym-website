# FitLife Gym Website

A premium, modern gym website built with React, Tailwind CSS, and Framer Motion.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features

-   **Responsive Design**: Fully responsive on mobile, tablet, and desktop.
-   **Modern UI**: Dark mode, glassmorphism, and neon accents.
-   **Animations**: Smooth transitions using Framer Motion.
-   **Pages**: Home, About, Programs, Trainers, Pricing, Contact.

## Technologies

-   React (Vite)
-   Tailwind CSS
-   Framer Motion
-   Lucide React
-   React Router DOM
